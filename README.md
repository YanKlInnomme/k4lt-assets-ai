# Module k4lt-assets-ai pour Foundry VTT

Ce module est intrinsèquement lié aux modules complémentaires (k4lt-fr, k4lt-en, k4lt-the-black-madonna et d'autres en devenir). Comme son nom le laisse supposer, il contient des ressources générées par IA. Celles-ci sont entièrement optionnelles et ne sont pas nécessaires au fonctionnement des autres modules. Voici une liste non exhaustive du contenu inclus dans ce module :

# k4lt-assets-ai Module for Foundry VTT

This module is closely tied to the companion modules (k4lt-fr, k4lt-en, k4lt-the-black-madonna, and others currently in development). As its name suggests, it contains AI-generated resources. These resources are entirely optional and are not required for the other modules to function. The following is a non-exhaustive list of the content included in this module:
 * Portraits and illustrations for pre-generated character sheets;
 * Illustrations for non-player characters (NPCs);
 * Atmospheric artwork for scenes, along with additional scenes designed to enrich the campaign;
 * A library of audio content designed to enhance immersion and support the pacing of different gameplay sequences;
 * Illustrations intended to enhance the appearance of documents and journal entries;
 * Additional visual and audio resources that will be added to future modules.

## AI-Generated Content

Please note that the AI-generated content:
 * Is unofficial and is not part of the original KULT content or its expansions;
 * Has not been approved, endorsed, or reviewed by Helmgast or the KULT rights holders;
 * Is provided free of charge as part of this module.

These resources were created to enrich the virtual tabletop experience while remaining clearly distinct from the canonical material. They follow their own artistic direction and narrative tone in order to avoid any confusion with the official KULT resources.

Following the changes to Foundry Virtual Tabletop's policy regarding AI-generated content, published in March 2026, these resources are no longer permitted in modules distributed through the official Foundry VTT platforms.

This module was therefore created to package these resources separately, allowing users who wish to do so to continue using them alongside the main modules.
